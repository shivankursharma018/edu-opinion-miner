# Contributing to Edu-Opinion-Miner

Want to help improve **Edu-Opinion-Miner**? Awesome! Here's all you need to know:

### 1. Fork & Branch

```bash
git clone https://github.com/shivankursharma018/edu-opinion-miner.git
git checkout -b feature/your-feature
```

### 2. Make Changes

* Update code, dashboard, docs, or data.
* Keep Python code clean (PEP8) and modular.
* Ensure dashboard renders correctly and plots work.

### 3. Test & Commit

```bash
git add .
git commit -m "Brief description of your change"
git push origin feature/your-feature
```

### 4. Pull Request

* Open a PR against `main`.
* Include screenshots or examples if relevant.

### 5. Report Issues

* Check existing issues first.
* Provide clear steps, logs, or screenshots.

---

Even small fixes or doc updates are valuable. Thanks for helping improve EduSense!
